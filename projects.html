Plateforme de services photovoltaïques

Développement d’une plateforme web dédiée à la mise en relation entre clients et prestataires de services photovoltaïques. Elle permet de gérer les profils des installateurs, afficher les catalogues, filtrer par région, et suivre les abonnements. Réalisée avec Symfony, Bootstrap, MySQL, et le moteur de template Twig pour une interface moderne et responsive.

USKE Manager – Plateforme de gestion d’équipe sportive

Application web réalisée pour le club de football USKE (Ksour Essef). Elle permet de gérer les joueurs, les matchs, les résultats, les actualités et le staff technique à travers une interface claire et responsive. Côté technique, le projet est basé sur le framework Symfony pour la logique métier, Twig pour le rendu des vues, Bootstrap pour un design moderne et responsive, et MySQL comme base de données relationnelle.

Devops Project

Une pipeline CI/CD automatisée pour une application Spring Boot et Angular, intégrant divers outils DevOps tels que Jenkins, SonarQube, Nexus, Docker, Grafana et Prometheus. Cette pipeline inclut des tests unitaires afin de garantir un déploiement fiable et sécurisé.

Passerelle de Paiement Électronique

Développement d’une passerelle de paiement modulaire basée sur une architecture microservices conteneurisée avec Docker, intégrant les solutions Stripe, PayPal et ClickToPay. La sécurité est assurée via Keycloak, et les performances sont surveillées avec Grafana pour garantir scalabilité et fiabilité.
